# Getiren · Akyaka

Yerel concierge/kurye uygulaması. **Güncel uygulama [`getiren/`](getiren/) altında** —
Laravel 13 + Vue (Inertia) + MySQL + Reverb, tamamı Docker'da.
Kurulum, servisler ve mimari için: **[`getiren/README.md`](getiren/README.md)**.

## Repo yapısı

- **[`getiren/`](getiren/)** — asıl uygulama (Laravel + Vue + Docker). Buradan başlayın.
- **`theme/`** — statik HTML tema taslakları (Version 1–3); Version 3 uygulamaya taşındı.
- Kök dizin — ilk vanilla-JS prototipi (bağımlılıksız demo), referans olarak duruyor:

```bash
npm start   # eski prototip · http://localhost:5173
```
